Arrays and Arraylists

What are Arrays?

An array is a collection of items stored in a singular data type, resembling a row of objects that can be modified at any time. However, it is essential to note that all the items in an array must be of the same data type. When initializing an array, its size is fixed and cannot be changed. Nevertheless, you can read the array and access any specific index to retrieve or modify its contents. Although the size of the array remains constant, the ability to manipulate individual elements provides flexibility and adaptability within the array structure.

An array is created in a manner similar to a class, with the size initialized inside the square brackets next to it. Additionally, we have the option to manually set the contents of the array using curly brackets. There are several operations we can perform on an array, one of which is traversing through its entire length. This can be accomplished using a for loop that iterates over each index in the array, allowing us to access and process the data stored at each position.

2D Arrays

Arrays in programming have the flexibility to store any data type, including other arrays. When an array stores other arrays, it creates a two-dimensional (2D) array. This transition from a single row of items to a square-like structure enables more complex data organization. 2D arrays can take on two forms: rectangular or ragged. In a rectangular 2D array, all rows have the same number of elements, forming a perfect square shape. On the other hand, a ragged 2D array allows rows to have different lengths, resulting in an irregular shape. To work with 2D arrays, specific syntax is used. Elements in a 2D array are accessed by using two sets of square brackets, where the first set indicates the row index, and the second set indicates the column index. This syntax allows programmers to manipulate and access values within 2D arrays.

What is an arraylist?

An ArrayList is an enhanced version of an array that offers the advantage of being resizable. Unlike traditional arrays, an ArrayList dynamically adjusts its size based on the number of elements it contains. This means that it can increase or decrease in size as elements are added or removed.

To declare an arraylist we write ArrayList<Data Type> where you put the data type you want to store between the angle brackets. However, primitive data types can no longer be stored and their wrapper classes must be stored instead such as Integer instead of int. Otherwise, you can treat it as a primitive data type.
