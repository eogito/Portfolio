Recurison

What is it?

Recursion is when you call the method inside of the same method. For example, if you have a multiplying array, you can call the multiply method again inside of it. In order to determine which iteration of the method is being run, you can think of the list of methods as a stack. When a method is called, that instance is added to the top of the stack. The program runs whatever method is at the top of the stack. In order to perform recursion, 2 things are needed:

A recursion step: this is where the method calls itself again.
